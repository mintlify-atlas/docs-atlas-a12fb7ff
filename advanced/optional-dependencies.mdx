## Overview

The Adopta Un Abuelo React Components library uses optional peer dependencies to keep the core bundle lightweight. Install only the dependencies you need based on which components you're using.

<Warning>
  Optional dependencies must be installed separately and are **required** for specific components to function. The component will not work if its dependencies are missing.
</Warning>

## Component Dependencies

### InputLocation Component

Required for location autocomplete functionality powered by Google Maps Places API.

**Package:** `@react-google-maps/api`

<CodeGroup>
```bash npm
npm install @react-google-maps/api
```

```bash yarn
yarn add @react-google-maps/api
```

```bash pnpm
pnpm add @react-google-maps/api
```
</CodeGroup>

**Additional Setup Required:** See [Google Maps Integration](/advanced/google-maps-integration) for API key configuration.

**Packages:** Multiple Tiptap extensions

<CodeGroup>

```bash yarn
yarn add @tiptap/react @tiptap/starter-kit @tiptap/extension-text-align @tiptap/extension-text-style @tiptap/extensions
```

```bash pnpm
pnpm add @tiptap/react @tiptap/starter-kit @tiptap/extension-text-align @tiptap/extension-text-style @tiptap/extensions
```
</CodeGroup>

**Additional Setup Required:** See [Rich Text Editor](/advanced/rich-text-editor) for configuration details.

## Dependency Matrix

Quick reference table:

| Component(s) | Required Packages | Additional Setup |
|--------------|-------------------|------------------|
| `InputLocation` | `@react-google-maps/api` | Google Maps API key |
| `Payout` | `@stripe/react-stripe-js`<br/>`@stripe/stripe-js` | Stripe publishable key |
| `TextArea` (rich text) | `@tiptap/react`<br/>`@tiptap/starter-kit`<br/>`@tiptap/extension-text-align`<br/>`@tiptap/extension-text-style`<br/>`@tiptap/extensions` | None |
| `Button` (animations) | `@lottiefiles/react-lottie-player` | None |
| `InputImage` | `react-webcam` | Camera permissions |

## Version Compatibility

All optional dependencies are tested with the versions specified in the library's `devDependencies`. Current tested versions:

- `@react-google-maps/api`: ^2.20.8
- `@stripe/react-stripe-js`: ^5.6.0
- `@stripe/stripe-js`: ^8.7.0
- `@tiptap/react`: ^3.19.0
- `@tiptap/starter-kit`: ^3.19.0
- `@tiptap/extension-text-align`: ^3.19.0
- `@tiptap/extension-text-style`: ^3.19.0
- `@tiptap/extensions`: ^3.19.0
- `@lottiefiles/react-lottie-player`: ^3.6.0
- `react-webcam`: ^7.2.0

<Note>
  The library follows semantic versioning. Patch and minor version updates of optional dependencies should be compatible, but test thoroughly when upgrading major versions.
</Note>
